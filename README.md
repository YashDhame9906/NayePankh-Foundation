# 🌱 NayePankh Foundation – NGO Website Redesign

A modern, responsive, and visually engaging NGO website redesign built for **NayePankh Foundation** using HTML, CSS, and JavaScript.

The project aims to create a better digital presence for the organization by focusing on emotional storytelling, modern UI design, smooth user experience, and mobile responsiveness.

---

## 🌐 Live Demo

[For Live Demo Click Here ](https://yashdhame9906.github.io/NayePankh-Foundation/)


---

## ✨ Features

###  Landing Hero Section

* Full-screen background image
* Emotional NGO-focused message
* Call-to-action buttons
* Transparent glassmorphism navigation bar

###  About Us

* Introduction to NayePankh Foundation
* Storytelling-based layout
* NGO history and purpose

###  Journey Timeline

* Timeline showing the organization's growth from COVID-19 initiatives to a youth-driven movement


###  Our Initiatives

* Real activity-based image cards covering:

  * Education support
  * Hunger relief
  * Menstrual hygiene awareness
  * Clothes distribution
  * Community welfare



###  Gallery

* Responsive image gallery
* Hover effects and visual storytelling

###  Contact Section

* Contact details
* Responsive contact form
